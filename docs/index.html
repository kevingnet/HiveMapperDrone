Hive Mapper — Drone Navigation

C++ command-line utility — shortest-path navigation on interconnected circular roads (Dijkstra).

Reads a world description file (roads, intersections, start, destination), computes minimum-time route, and can emit a Graphviz .dot file.

Commands: GO, STOP, REVERSE, TRANSFER

Run locally:

make
./bin/drone_navigator -a dijkstra -f test_cases.txt
